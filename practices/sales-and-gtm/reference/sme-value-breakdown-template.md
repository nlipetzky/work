# SME Value Breakdown — Template

Practice-level, entity-agnostic. The **outbound** mirror of the Discovery Snapshot. Where the
snapshot maps a client's current state, this maps an expert's market value so we can run
campaigns that book meetings *for the expert*.

Premise (see memory `sme-partner-credibility-model`): we sell access to the human, not the AI.
The campaign's job is to get a target segment to say yes to a meeting with the expert. One
expert has *several* value props ... run one sharp campaign per prop, not one muddy "they do AI."

Copy this into the partner's folder (`accounts/ventures/<entity>/` or
`accounts/prospects/<partner>/`) as `sme-value-breakdown.md` and fill it.

## Value-prop inventory

Decompose the expert into distinct reasons a segment would want to talk to them. Separate
**credentialed-specialist** props (narrow, high-authority) from **generalist/operating** props
(domain-agnostic, often the most scalable).

| # | Value prop | Type (specialist / operating) | Target segment | Maps to offer | The promise (outcome) | Liability tier |
| --- | --- | --- | --- | --- | --- | --- |
| 1 |  |  |  |  |  | low / high |

Liability tier matters: we monetize the expert's credibility, so AI output under their name is
what can burn it. Favor **low-liability, high-leverage** props for the agentic offer; keep
high-stakes regulated work human-led or out of scope.

## Selection note

Which props to lead with and why. Rank by: clarity of segment × strength of the promise ×
lowest liability. A prop with a fuzzy segment or a weak promise is not campaign-ready ... park it.

---

## Definition of Ready (campaign-launch gate)

A value prop is ready to build a campaign around when:

- [ ] **Named, reachable target segment** (who, where we find them).
- [ ] **A concrete promise** the meeting delivers (not "learn about AI").
- [ ] **A hook** rooted in the expert's POV/proof, not our tooling.
- [ ] **The offer it routes to** is identified (managed-agent shape).
- [ ] **Liability tier assessed** and acceptable for AI-assisted delivery under the expert's name.

Missing any → the prop isn't campaign-ready. Sharpen or park it.
