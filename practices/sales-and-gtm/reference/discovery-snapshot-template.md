# Discovery Snapshot — Template

The milestone knowledge-state for a partner-sourced opportunity. Filled by the **partner** in
plain language during/after the free diagnostic. When the Definition-of-Ready checklist at the
bottom is fully answered, the opportunity is handed to Nick to build. See
`partner-led-discovery-protocol.md` for the motion and the gate rules.

Copy this file into the opportunity's folder as `discovery-snapshot.md` and fill it.

## Definition of Ready  (the gate)

Hand to Nick only when ALL are answered. Any blank → back to the partner.

- [ ] **Single highest-impact bottleneck named.**
- [ ] **Smallest first intervention defined, with a concrete promised result.**
- [ ] **Systems of record involved identified** (what we touch vs leave alone).
- [ ] **Guardrails / off-limits named.**
- [ ] **Offer shape confirmed = managed agentic capacity, not custom software.**
- [ ] **Decision-maker identified and interested in proceeding** (build-on-demand trigger).
